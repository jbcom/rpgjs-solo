# create-rpgjs

CLI officiel pour initialiser un projet RPGJS.

## Usage

```bash
npm create rpgjs@latest
```

Options:

```bash
npm create rpgjs@latest my-game -- --template base
npm create rpgjs@latest my-game -- --template-url https://github.com/org/rpgjs-template.git
```

## Local

```bash
pnpm -F create-rpgjs build
node ./packages/create-rpgjs/dist/bin/create-rpgjs.js
```
