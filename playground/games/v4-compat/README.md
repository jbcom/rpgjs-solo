# V4 Compatibility

Compatibility flow for RPGJS v4-style configuration, modules, maps, and GUI
packages.

## Run

From the repository root:

```bash
pnpm playground
```

Then select **V4 Compatibility** in the launcher at `http://localhost:5174`.

Run only this game:

```bash
pnpm --dir playground dev:v4-compat
```

Direct URL: `http://localhost:5184`.
